## 📘 Descripción

Este repositorio contiene las prácticas realizadas durante mi formación en *Big Data con Scala*.  
Aquí recopilo diferentes ejercicios enfocados en programación funcional, manejo de colecciones y procesamiento de datos.

El objetivo de este repositorio es:
- Reforzar conceptos clave de Scala 🧠
- Aplicar técnicas de programación funcional ⚙️
- Prepararme para trabajar con herramientas de Big Data como Spark 🚀

---

## 📂 Contenido

En este repositorio encontrarás:

- Ejercicios con *List, Array y Vector*
- Uso de funciones de orden superior (map, filter, reduce)
- Manipulación de Strings
- Resolución de problemas prácticos paso a paso
